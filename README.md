## ğŸ’» Getting Started

### Prerequisites
- Node.js (v18+)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Sajimithan/Ailabs.git
   ```
2. Navigate to the frontend directory:
   ```bash
   cd ai-labs-glass
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
